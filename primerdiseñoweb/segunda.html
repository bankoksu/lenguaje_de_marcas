que te validesa XHTML

XHTML (Lenguaje de Marcado de Hipertexto Extensible) es una versión más estricta y limpia de HTML , que nace precisamente con el objetivo de remplazar a HTML ante su limitación de uso con las cada vez más abundantes herramientas basadas en XML . XHTML extiende HTML 4.0 combinando la sintaxis de HTML, diseñado para mostrar datos, con la de XML, diseñado para describir los datos.

que es estrito?

El estándar XHTML deriva de XML, por lo que comparte con el muchas de sus normas y sintaxis. Uno de los conceptos fundamentales de XML es la utilización del DTD o Document Type Definition ("Definición del Tipo de Documento") en resumen buscas unificar este lenguaje.

que es transitional?

Este es el DTD Transicional de HTML 4.01, que incluye los atributos y elementos de presentación que el W3C espera que dejen de utilizarse a medida que madure el soporte de las hojas de estilo
